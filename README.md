# ubuntu-autoinstall

Custom Ubuntu 24.04 LTS autoinstall for testing purposes.

Desktop Environment: KUbuntu

Base image: Ubuntu Server 24.04 LTS

Unprivileged user credentials:
- username: `user`
- password: `user`

Privileged user credentials:
- username: `admin`
- password: `admin`

- docker-ctop (azlux.fr repo)
